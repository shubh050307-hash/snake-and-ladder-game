This is a college level project of snake and ladder game made using html , tailwind css and vanilla js
This is a game made completely on web using the layer of gemini 
Features:- 
1 - takes inputs from the user with the help of a single click
2 - after the click the dice rolls 
3 - randomly selects a number from 1-6 
4 - the number which lands finally the user moves accordingly 
5 - when the user comes near the face of the snake the snake bites and the user is redirected to the bottom lenght of the snake 
6 - user then starts from that position
