# 🎲 Snake & Ladder Game (HTML, CSS, JavaScript)

A fun and interactive **Snake & Ladder game** built using **HTML, CSS, and Vanilla JavaScript**. This project recreates the classic board game experience in the browser with smooth UI, dice rolls, and player movement logic.

---

## ✨ Overview

Snake & Ladder is a timeless board game where players race to reach the final square while avoiding snakes and climbing ladders. This web-based version brings that experience online with dynamic gameplay and a clean interface.

---

## 🚀 Features

* 🎯 **Classic Gameplay**
  Play the traditional Snake & Ladder game with standard rules.

* 🎲 **Dice Roll Simulation**
  Random dice generation for fair gameplay.

* 👥 **Two-Player Mode** *(or more if implemented)*
  Play with friends on the same device.

* 🐍 **Snakes & Ladders Logic**

  * Climb ladders to move ahead
  * Slide down snakes when landed on

* 🎨 **Interactive UI**
  Visually appealing board with smooth movements.

* 📱 **Responsive Design**
  Works on desktop and mobile browsers.

## ▶️ How to Play

1. Open the game in your browser
2. Click the **Roll Dice** button
3. Move your player based on the dice value
4. Follow the rules:

   * Landing on a **ladder** moves you up
   * Landing on a **snake** brings you down
5. First player to reach the final tile wins 🎉

## 🧠 How It Works

* JavaScript generates a random number (1–6) to simulate dice rolls
* Player positions are tracked using variables
* The board is mapped with predefined snake and ladder positions
* DOM manipulation updates player movement visually
